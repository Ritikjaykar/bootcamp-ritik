Drill 1: Task Audit

Task: "Find the latest press release"

Ideal 5-Step Flow:

  1. User lands on homepage
  2. Sees clear "News" or "Press" section
  3. Clicks on press section
  4. Views list of press releases sorted by date
  5. Clicks on most recent release to read

Current Friction Points:

  • No dedicated press section in navigation
  • Press releases mixed with blog posts
  • No clear date sorting
  • Small click targets on mobile
  • Slow loading of news images

Proposed Fixes:

  • Add "Press" to main navigation
  • Create separate press release section
  • Sort by publication date (newest first)
  • Increase touch target sizes
  • Optimize and lazy load images
